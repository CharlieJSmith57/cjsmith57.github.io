HEADSHOT UPLOAD INSTRUCTIONS
=============================
1. Name your photo file exactly: headshot.jpg  (or headshot.png)
2. Drop it into this folder: assets/images/
3. The about page will automatically show it in the circular avatar.

Recommended specs:
  - Square crop (1:1 ratio), e.g. 400x400px or 600x600px
  - JPG or PNG
  - File size under 500KB for fast loading

The about page already has this code in place:
  <img src="assets/images/headshot.jpg" ... />
  with a CJS text fallback if the file is missing.
