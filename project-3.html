6-DOF Robotic
Gripper

Year 2023
Role Mechanisms Designer
Duration 5 months
Industry Industrial Automation

Tools Used

  • SolidWorks (mechanisms)

  • MATLAB / Simulink
  • Python (control)

Design Methods

  • Compliant Mechanism Theory
  • Pseudo-Rigid-Body Model

  • DFM for Additive Mfg.

Materials

  • PA12 (SLS Nylon)

  • Titanium (fasteners)

Overview

Automated assembly lines handling fragile electronics or soft goods require grippers that can adapt to irregular geometries and self-limit applied force — without requiring force sensors or complex closed-loop control. This project developed a 3-finger, 6-degree-of-freedom compliant gripper designed around passive mechanical intelligence.

The key design challenge: grip objects ranging from 20 mm to 120 mm in diameter with contact forces never exceeding 3 N, using only a single linear actuator input — no active sensing.

Approach

I used pseudo-rigid-body modeling (PRBM) to synthesize the compliant mechanism geometry. Each finger consists of a bistable, monolithic flexure printed in PA12 nylon, with a TPU 95A insert at each joint to tune stiffness independently. The three-finger assembly uses a differential pulley mechanism on the actuation cable so force distributes passively across all fingers regardless of object geometry.

Kinematic synthesis set the finger profiles so the gripper self-centers on circular and rectangular objects automatically. ANSYS large-deformation FEA validated the flexure fatigue life — the design target was 500,000 cycles minimum.

A rapid prototyping loop with 8 physical iterations refined the joint stiffness profile, which proved sensitive to print orientation. Final parts were SLS-printed in a single build for each finger, eliminating all assembly at the joint level.

<3N Max Contact Force
500k+ Cycle Fatigue Life
20–120 mm Grip Range

Outcome

The prototype achieved all performance targets in bench testing: grip force consistently under 3 N across the full size range, passive self-centering within 2 mm for symmetric objects, and zero grip failures across 500,000 cycles in accelerated life testing. The monolithic finger design reduced part count by 74% compared to a comparable rigid-link gripper with equivalent DOF.

The gripper was integrated into a production pilot line where it replaced a pneumatic suction system that had 12% cycle failures on irregular package geometries. Failure rate dropped to under 0.4%.
