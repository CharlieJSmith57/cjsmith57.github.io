Overview

This portfolio site is itself a project. I built it using Claude Code by Anthropic, iterating on the design and content via prompt engineering and using git locally on my laptop. The result is a fully custom static site with no frameworks or build tools: just HTML, CSS, and vanilla JavaScript, hosted for free on GitHub Pages.

The goal was to create a site that felt professional and custom without relying on templates, able to showcase my experience and accomplishments. Every page, layout decision, and design detail was chosen and developed through multiple rounds of iteration.

Development Process

The site was built iteratively using Claude Code, starting from a design brief (dark & sleek, mechanical engineering project-based focus) and evolving through conversation. Pages were built one at a time (home, project list, about page, individual project write-ups) with each adding new components and CSS patterns.

Content was grounded entirely in real information: actual projects, real work experience, accurate coursework. As the site developed, real documents like a resume PDF were integrated, and features like embedded PDF viewing and a headshot photo were added on request.

Version control was maintained throughout via Git, with the final site deployed to GitHub Pages from the main branch making updates as simple as pushing a commit.

Design System

The design uses a minimal set of CSS custom properties as a token system: one background color scale, one accent color (lime green, #c8ff00), and three font families: Bebas Neue for display headings, DM Sans for body text, and DM Mono for labels, tags, and metadata. This keeps the visual language consistent across all pages without a CSS framework.

The layout is fully responsive: a hamburger nav on mobile, single-column stacking for the about grid and project body, and fluid typography using clamp() for headings.
